ProtoCaller.Wrappers.biosimspacewrapper module
==============================================

This is the module which adds some extra functions on top of BioSimSpace which
are used during different parts of the code. Here it is possible to centre a
system in a box and resize the dimensions of the box. More advanced functions
allow REST2-like rescaling and dummy bond rescaling for the final lambda value
of the perturbation.
